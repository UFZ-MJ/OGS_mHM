# mHM\#OGS #

mHM\#OGS is a coupled hydrological model which couples the mesoscale Hydrological Model 5.7 (mHM5.7) and OpenGeoSys5 (OGS5). Tne interface GIS2FEM is used to communicate between those two codes.
